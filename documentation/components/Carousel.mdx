# Carousel

```javascript
import { Carousel } from "mfcl";
```

## Description

The Carousel component allows you to a display a selection of rotating items.

## Properties

<Props of={Carousel} />

## Basic Usage

<Playground>
  <Carousel ariaLabel="one">
    <div style={{ backgroundColor: "#0170ad", color: "#fff" }}>Slide 1</div>
    <div style={{ backgroundColor: "#17aada", color: "#fff" }}>Slide 2</div>
  </Carousel>
</Playground>

## Responsive Breakpoints

The Responsive prop overrides the provided props of the carousel at different pixel breakpoints

<Playground>
  <Carousel
    ariaLabel="one"
    itemsToShow={2}
    layoutGap={5}
    responsive={[
      {
        breakpoint: 700,
        itemsToShow: 1,
        hideIndicators: true,
        hideControls: true,
        layoutGap: 0,
      },
    ]}
  >
    <div style={{ backgroundColor: "#0170ad", color: "#fff" }}>Slide 1</div>
    <div style={{ backgroundColor: "#17aada", color: "#fff" }}>Slide 2</div>
    <div style={{ backgroundColor: "#E5E5E5", color: "#000" }}>Slide 3</div>
    <div style={{ backgroundColor: "#2b0b3b", color: "#fff" }}>Slide 4</div>
  </Carousel>
</Playground>

## Control Button Alignment

Any non conflicting combination of 'top', 'middle', 'bottom', 'left', 'right', 'center'.  
hideControls prop removes the control buttons from view until hovered.
